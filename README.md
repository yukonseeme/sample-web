# Rabbit Energy Website Project

## Overview
The Rabbit Energy website serves as a comprehensive platform for users to explore energy-saving solutions and products. The goal of the project is to educate users about energy efficiency while providing tools and resources for reducing energy consumption.

## Features
- **Home Page**: Welcoming users with an overview of energy-saving solutions.
- **Products Page**: Listing various energy-efficient products.
- **Resources**: Articles and guides on energy conservation strategies.
- **Contact Us**: A form for users to reach out with questions or feedback.

## Technologies Used
- **Front-End**: HTML, CSS, JavaScript
- **Back-End**: Node.js, Express
- **Database**: MongoDB
- **Frameworks**: React for the front end, Bootstrap for styling

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yukonseeme/sample-web.git
   ```
2. Navigate to the project directory:
   ```bash
   cd sample-web
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
After following the installation steps, open your web browser and go to `http://localhost:3000` to view the Rabbit Energy website in action.
